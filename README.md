# Hypermedia project. Part 1
He distribuido el curriculum web de forma que al inicio se vea una imagen mia con la informacion inportante como nombre, correo y telefono mas una breve descripcion de mi. A la derecha de mi persona hay un slider donde se mostraran los projectos mas inportantes en los que he participado, cada uno con un link a dicho projecto.
Al bajar se vera un acordeon donde se desplegara la formacion y experiencia laboral con un rango de flecha del inicio a la ultima actualizacion, seguidamente los idiomas que hablo y el nivel de fluidez.
Seguidamente se ven dos columnas, a la derecha estan las habilidades mas destacables con una barra que representa el nivel y a la derecha hay un grafico que muestra las soft skills con un rango del 0 al 100.
Y por ultimo pero no menos inportante un portfolio de todos los projectos con una imagen que al hacer clic lleva al github del projecto.
